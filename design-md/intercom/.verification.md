### Single-system: 6px hero / 4px compact + Cream canvas + Charcoal Primary

Intercom uses a **two-tier radius system + dual-canvas dual-product chrome**:
- **Hero Primary**: 6px / 42px / `#000` or `#fff` (canvas-inverted on fin.ai dark hero)
- **Compact Primary**: 4px / 40px / `#111111` Intercom Charcoal
- **Canvas**: `#faf9f6` Cream (warm-print) on intercom.com; dark on fin.ai
- All weight 400 — never bold; size + cream canvas warmth carry the visual register
- **Two product surfaces, unified Primary type**: intercom.com helpdesk + fin.ai AI Agent each have their own canvas but the same Charcoal/White Primary CTA pattern

This confirms the §11 "warm-cream / craft" register and the dual-product 2024-2025 Fin spinout positioning.
