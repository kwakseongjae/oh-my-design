### Single-system: 8px Near-Black Primary + Lovable Violet featured + Cream `#fcfbf8` warmth

Lovable uses a **disciplined 8px-radius binary chrome with violet accent for featured contexts**:
- Primary CTA: `lab(0 0 0 / 0.88)` (Near-Black 88%) 8px / 32px / 6×10 / 14px·400
- Featured Primary: **Lovable Violet `lab(47.92 57.95 -81.30)`** (~`#6038c8`-ish vivid bluish-purple) 8px / 36px — only on pricing-featured tier
- Cream `#fcfbf8` 6px selector chips — warm off-white for in-tier UI controls (1 step warmer + 2px less radius than Primary)
- Nav 0px text-only ghost / 15px·400 (1px larger than CTAs — readability priority)
- All weight 400 — never bold; the canvas warmth + violet accent carry the visual register

The use of `lab()` color space throughout (not just `#hex` or `rgb`) is unusual and notable — Lovable uses **CSS Color Level 4 lab()** as canonical token expression, suggesting modern CSS-first DS practices. The `lab(0 0 0 / 0.88)` Primary is intentional translucency (88%) — gives a soft true-black that maintains contrast against any canvas tint.

This corrects the prior footer's claim of "Sign in `rgba(0,0,0,0.05)` 8px" — that's a different (Outline-light) variant. The canonical Primary is `lab(0 0 0 / 0.88)` Near-Black with featured Violet variant.
