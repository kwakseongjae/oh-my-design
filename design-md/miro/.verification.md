### Single-system: 8px Three-Color Primary

Miro uses an **8px-radius three-color CTA system** rare in the corpus:
- **Yellow `#fde050`** — banner promos only, **weight 900 ALL CAPS** (`#1c1c1e` text, the only Black weight)
- **Charcoal `#1c1c1e`** — default Primary, weight 600 (white text)
- **Miro Blue `#3859ff`** — featured-pricing-tier accent (white text, weight 600)
- All radii **8px** (consistent), no pill chrome anywhere
- Heights tier: 40 (banner) / 42-44 (hero) / 48 (pricing-large)
- Weight system: **600 default** with **900 ONLY on Yellow banner** — the heavy weight is itself a signal

This three-color discipline maps to Miro's positioning rhythm: brand-attention (Yellow) → conversion (Charcoal) → upsell (Blue). Each color earns its appearance.
