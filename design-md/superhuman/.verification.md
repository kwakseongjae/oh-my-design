### Two-system: Lavender Suite + Sky Blue Mail (post-merger split)

- **Unified Suite (superhuman.com home, post-July-2025)**: **Lavender Glow `#d4c7ff`** Primary 8px / 36px / 14px·**600** + **Lavender Dark `#714cb6`** Outline-text variant + **Cream `#ece9e2`** sub-nav (via `color(srgb)`) + **Maroon `#421d24`** cross-promo banner — refined "luxury" register

Distinctive **weight 460** appears in 3+ places — between regular (400) and medium (500), unusual intermediate weight that preserves the "luxury-quiet" register.

**Color-space innovation**: uses `color(srgb 1 1 1 / 0.95)` and `color(srgb 0.925383 …)` — modern CSS Color Level 4 explicit-space syntax (less common than oklab/lab but still wide-gamut-aware).
