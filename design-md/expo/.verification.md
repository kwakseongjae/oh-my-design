### Single-system: Two-tier pill chrome + 8px content cards

Expo uses a **height-equals-radius pill** convention (radius = height = full pill):
- Hero Primary: 48px height + 48px radius / 0×24 / 16px·500
- Header + Pricing CTA: 36px height + 36px radius / 0×16 / 14px·500
- Pricing nav pill: 36px height + 24px radius (only deviation — slightly less round to differentiate active-state nav from action CTAs)
- Testimonial cards: 8px content radius (separate visual track)

Two fills: Black `#000` Primary + Cool Gray `#f0f0f3` Secondary. Near-black text uses `#1c2024` (warm-blue near-black, not pure `#000`) on light surfaces.
