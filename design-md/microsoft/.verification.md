## Font evidence

### Live computed, FontFaceSet, and source corroboration

- `Segoe UI Variable Text` is **loaded / high confidence**, with 797 visible uses across badges, body, buttons, cards, dialogs, headings, menus, tabs, and text. The collector ties computed use to loaded FontFace entries and five Microsoft-hosted Segoe Sans source families, including `https://www.microsoft.com/echo/etc.clientlibs/store/clientlibs/clientlib-reimagine/page/base/resources/fonts/segoe-sans/west-european/latest.woff2`.
- `SegoeUI` is **loaded / high confidence**, with 242 visible uses in public-web chrome and content. The collector records Microsoft-hosted normal, light, semilight, semibold, and bold sources under `https://c.s-microsoft.com/static/fonts/segoe-ui/west-european/`.
- `Segoe UI Variable Display` is **loaded / high confidence**, with 58 visible heading uses, using the same Microsoft-hosted Segoe Sans asset domain. `Segoe UI Variable Small` is loaded with only two visible text uses and is not generalized beyond that observation.
- Resolution: `Segoe UI Variable Text`, `Segoe UI Variable Display`, and `Segoe UI` are the sole typography-family tokens, each limited to its observed role. This conclusion requires the computed family, loaded FontFace status, and source URL together.

### Official asset and licence boundary

- Microsoft describes Segoe UI as its screen-oriented branding typeface; Fluent identifies Segoe UI as its primary web typeface and Segoe UI Variable as its Windows ramp face.
- Microsoft’s font redistribution FAQ permits naming Windows fonts in a CSS stack but prohibits self-hosting Windows font files from an installation. It says Segoe UI may be licensed through Monotype, while Segoe UI Variable is not available for licensing or use outside Microsoft products or on non-Windows platforms.
- The captured Microsoft-hosted files establish use on the observed pages; they are not reusable project font assets.

### Declared-only, system, and unresolved families

- **Declared-only:** Cascadia Code (three Fluent documentation source URLs, zero visible use); FabricMDL2Icons; MWF-FLUENT-ICONS; Leelawadee UI Web; regional Segoe UI Web declarations; and SegoeUI Fallback. None becomes a UI-family token.
- **Domain boundary:** no authenticated Microsoft 365 application, Windows application, Store checkout, Azure portal, or product-specific documentation surface was captured. Public webfont evidence does not populate those domains.
