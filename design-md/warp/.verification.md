### Single-system: 3px Cream + warp `oklch()` color-space + pill toggle exception

Warp uses **the most advanced color-space convention in the corpus** — `oklch()` for solid colors + `oklab()` for translucent variants:
- All Primary CTAs: **3px** radius (sharper than typical 4-6px — terminal-IDE register)
- **Warp Cream `oklch(0.97 0.01 84.6)`** + **Warp Espresso `oklch(0.14 0.004 84.6)` text** — both in **OKLCH** (Lightness/Chroma/Hue) color space, the most perceptually-uniform CSS Color Module Level 4 syntax
- **`oklab()` for translucent**: e.g., billing toggle uses `oklab(0.97 ... / 0.1)` for 10% transparency
- **Outline radius 4px** (sub-tier 1px more rounded than Primary 3px — subtle distinction)
- **Billing toggle 9999px** full pill (the only round chrome — appropriate for binary state switch)
- All weight 500 — never bold

**Color-space ranking in corpus** (most advanced first):
1. **Warp** — `oklch()` + `oklab()` (perceptually-uniform LCh + Lab) ← HIGHEST
2. Sanity — `display-p3()` (wide-gamut)
3. Mistral — `oklab()` (perceptually-uniform Lab only)
4. Cursor / Lovable / Mintlify — `lab()` (CIE Lab)
5. Most others — `rgb()` / `#hex`

This refines the prior footer's "nav `#242526` 0px / 8×16 / 95px / 16px·400" — values appear to be from a different surface inspection (different page state). Tier 1 confirms canonical **3px Cream Primary** with `oklch()` tokens.
