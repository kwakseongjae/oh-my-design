### Single-system: 4px Resend Blue Primary + 16px tab pills + 9999px banner

Resend uses a **three-radius scale**:
- **Primary CTA**: `#00a3ff` Resend Blue **4px** sharp / 41px / 12×20 / 14px·**600** (Hero CTA — distinctive 4px not pill)
- **Tab pills**: `rgba(24,25,28,0.88)` Charcoal-translucent **16px** / 40px / 0×16 / 14px·600 (active pricing-page tab)
- **Announcement banner**: 9999px full-pill `rgba(211,237,248,0.114)` translucent blue (top promo only)
- **Outline ghost**: 16px (hover-fill same as tab pill)
- All weight 600 — matches §11 "premium-developer" register
- Mid-Gray `#a1a4a5` for nav text

**Critical correction**: prior footer claimed "pill 16px / 0×16-20 / 40-48px / 14-16px·600" — that captured the **16px tab/ghost variant** but missed the **canonical Primary CTA which is `#00a3ff` Resend Blue at 4px** (the brand-defining color + radius). Major correction.
