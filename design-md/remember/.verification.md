## Methodological caveats

- The shared playwright browser session was concurrently used by sibling reference builds (`wanted`, `zigzag`, `ably`, `kakaopay`, `29cm`, `gangnamunni`, `zigbang`). Tab focus jumped repeatedly. Mitigated with explicit `browser_tabs.select(0)` and a `location.href.includes('remember')` guard in every `evaluate` call. The final successful payload confirms `url: "https://career.rememberapp.co.kr/job/postings"`.
- The Storybook iframe at `dramancompany.github.io/remember-ui/` lazy-loads story content via JS. Body-root tokens (bg `#F3F2EF`, Pretendard) captured; story-specific computed styles not captured because iframe focus was lost to concurrent tabs. Acceptable: career surface provided primary-component coverage.
- No Tier 2 fallback was available, so §4 carries no Tier 2 → Tier 1 conflict resolutions. Footer documents this honestly per skill rule "거짓 주장 금지 / 충돌 silent 해결 금지."
