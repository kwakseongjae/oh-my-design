## Official component evidence

### Box button

The official page defines solid green, solid red, outline green, and outline gray types. States are active, inactive, pressed, loading, on, and off. Official SVG assets provide 29–30px small, 36px medium, 43px large, and 48px extra-large examples with 5px or 6px radii.

### Capsule button

The official page defines active and pressed states, top/bottom scroll placement, disappearance after reaching the target content, optional avatar/thumbnail content, and a prohibition on arbitrary customization.

### Input

The official page defines required underline, text area, reset action, optional label/required mark/helper/counter, and password visibility control. State families cover text, password, and code inputs. Focus uses green; error changes underline, cursor, and helper text to red.

### Popup

The official page defines content, optional content, action, and scrim areas; vertical and simple layouts; one to three actions; disabled affirmative action; and one-popup-at-a-time behavior.

### Tabs and badges

Tabs define selected, unselected, and pressed states plus text/box/icon types. Badges define N, dot, and number variants, with green outside GNB and red reserved for GNB placement.
