### Single-system: 5px CTA / 4px nav + per-product accent

HashiCorp uses **one button geometry with per-product accent**:
- Shared chrome: `#1060ff` HC Blue Primary, `#fafafa`/`#3b3d45` Light Secondary, all **5px** radius / **500** weight
- Product-page Primary swaps to product-specific accent: **Terraform `#7b42bc` Purple** (others to verify per product)
- Two height tiers: 36px (header/inline) / 48px (hero/page-CTA)
- Nav items use **4px** (1px less than CTAs) for intentional sub-hierarchy distinction
- Padding: 9×15 (compact) / 11×19 (hero) — proportional

This is a sophisticated multi-product DS pattern — shared chrome with per-product accents, all rooted in the same 5px / 500-weight / Charcoal-text discipline.
