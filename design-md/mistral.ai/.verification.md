### Single-system: 0px Sharp + oklab Color-Space Canonical

Mistral AI uses an **all-0px-radius chrome** with **`oklab()` color-space token convention**:
- Every CTA + nav: **0px sharp** — no rounding anywhere (Parisian-precision register)
- Primary `#1f1f1f` Mistral Charcoal — warm-cast near-black
- Secondary uses **canvas-aware oklab** variants:
  - On dark canvas: `oklab(0.999994 ... / 0.1)` translucent White (pure-white in oklab perceptual space)
  - On light canvas: `oklab(0.955114 -0.00188443 0.0614298 / 0.5)` warm-cream (signature golden cast — `b` channel `0.0614298` shifts toward yellow)
- Cream `#fff0c2` carousel-next chrome confirms the warm-cream brand thread
- All weight 400 — never bold; the radius + canvas warmth carry the visual register

The use of **oklab() color space** is the Tier 1 signature (alongside Cursor + Lovable + Mintlify but Mistral is the first observed using **`oklab()` not `lab()`** — `oklab` is perceptually-uniform). This is one of the most modern CSS Color Level 4 adopters in the corpus.

1. **oklab() canonical convention** as DS pattern (more advanced than lab())
2. Canvas-aware oklab variants (dark vs light hero)
