## Logo decision

- Google favicon proxy (`https://www.google.com/s2/favicons?domain=dropbox.com&sz=128`) returned 332 bytes of HTML (error page, not an image); `www.dropbox.com` variant 336 bytes HTML — proxy unusable for this domain.
- `https://cdn.simpleicons.org/dropbox` → HTTP 200, valid SVG (fill #0061FF) → **logo: type simpleicons, slug dropbox**.
