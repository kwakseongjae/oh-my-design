The collector correlates its visible computed family with loaded FontFace entries and fourteen Supabase-hosted WOFF2 URLs split across `frontend-assets.supabase.com/www/...` and `frontend-assets.supabase.com/docs/...`.
- **Manrope:** loaded/high, 150 visible uses across body, buttons, and h1–h4. The collector correlates it with twelve Supabase-hosted WOFF2 URLs. The home h1 is 46px/500/46px and is retained as a route-local display sample.
- **Source Code Pro:** loaded/high, 13 visible uses across body, button, and h3 roles, with seven Supabase-hosted WOFF2 URLs. It remains a live public family but the packet does not establish a complete code or product-editor scale.
- **Resolution:** Inter, Manrope, and Source Code Pro are the only `tokens.typography.family` values because each has computed visible use, loaded FontFaceSet corroboration, and supplied source URLs. `ui-sans-serif` is system-only and is excluded.

### Official font and licence boundary

- Official Inter source and licence: https://github.com/rsms/inter — Inter is published under the SIL Open Font License.
- Official Manrope source and licence: https://github.com/davelab6/manrope — the project identifies Michael Sharanda as designer and provides the SIL Open Font License boundary.
- Official Source Code Pro source and licence: https://github.com/adobe-fonts/source-code-pro — Adobe’s repository describes it as a monospaced UI/coding family under OFL-1.1.
- Supabase-hosted WOFF2 URLs corroborate public-use delivery, not permission to redistribute those exact hosted files. A downstream target must use its own legitimate font source; no system fallback may be presented as one of these named families.
