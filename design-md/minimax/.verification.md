### Single-system: Charcoal Pill + Ghost Model Tabs (3-tier radius)

MiniMax uses a **modular-product-family chrome** with three radius tiers:
- **Header Primary**: `#181e25` Charcoal **32px** rounded-rect / 38px / 16px·400
- **Page Primary**: same Charcoal but **9999px** full-pill / 35px / 16px·400 (release-page register slightly more product-launch-energetic)
- **Model nav tabs**: ghost **12px** / 40px / 16px·400 — the signature MiniMax UI element (each model gets its own tab in a horizontal rail)
- **Cookie utility**: 8px / weight-600 — separate utility track

The **3-radius scale (32 / 12 / 9999)** + 8px utility = 4 discrete hierarchies. All weight 400 except cookie banner. The model-tab pattern reflects MiniMax's brand position: a multi-model AI lab where every model has its own product page (M2.7 / M2-Her / Speech 2.x / etc).

This corrects the prior footer's claim of "model nav tabs 12px / 8×24 / 40px / 16px·400 ghost" — that's accurate but missed the **canonical Header Primary `#181e25` 32px** + **Page Primary 9999px** patterns entirely.
