### Single-system: 4px Mercari Red Primary + 18px filter pills

Mercari uses a **disciplined three-tier radius system**:
- **Primary CTA**: `#ff333f` Mercari Red 4px / 11×15 / 45px / 15px·**700** (BOLD register matching karrot/freee East-Asian convention)
- **Outline + Header utility**: white 4px / 36px / 14px·400 + Charcoal text `#333333`
- **Search filter chips**: 18px full pill / 4×8 / 36px / 14px·400 (lighter weight, sub-CTA hierarchy)
- **Category tabs**: 0px text-only / 14px·**700** (active-tab indicator color shift to Mercari Red)
- US-locale promo banner uses **Bright Blue `#0095ee`** as a contextual accent (cross-border surfaces only — not the canonical brand red)

This dual-color usage (Mercari Red for JP, Bright Blue for US-promo) reflects Mercari's bicultural positioning: Japan-first brand identity + dedicated US-locale accent for the 2014-launched US arm.
