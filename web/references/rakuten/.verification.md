## Notes

- ReX guideline site (rex.rakuten.design / Brand & ReX Guidelines at global.rakuten.com/corp/brand/) is **NDA-gated**; GitHub org rakuten-rex states "no longer available after April 1st 2021". The npm tarballs remain the only public official component spec — treated as Tier 1 (brand-owned) but timestamped 2020–21.
- ranking.rakuten.co.jp served the access-concentration page to headless chromium (twice); books.rakuten.co.jp accepted a real-UA ja-JP context and serves as the second commerce surface.
- Logo: Google favicon proxy is broken for rakuten.co.jp AND www.rakuten.co.jp (334B/338B HTML error documents, not images). `https://cdn.simpleicons.org/rakuten` → HTTP 200 → `logo: simpleicons/rakuten`.
- Philosophy sources: corp.rakuten.co.jp/about/philosophy/ + /principle/ (WebFetch verified), Rakuten Font / Tone of Voice via Dalton Maag + global.rakuten.com font pages (WebSearch).
