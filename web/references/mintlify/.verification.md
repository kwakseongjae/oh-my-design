### Single-system: 9999px Pill discipline + lab() canonical tokens

Mintlify uses an **all-pill chrome** with `lab()` color-space token convention:
- Every CTA + nav item: **9999px full-pill** (`3.35544e+07px` literal — same `2^25` convention as Cursor — guaranteeing pill rendering at any height)
- Two height tiers: header/utility 32-34px / pricing-hero 40px
- Two fills: `lab(100 0 0)` Pure White and `lab(2.42579 -0.165291 -0.470081)` Mintlify Near-Black (the `lab()` value's `b` channel `-0.47` gives a subtle blue cast — characteristic of premium dark-tech interfaces)
- All weight 500 (UI medium)
- Padding scales: 4×12 (nav) / 4.5×12 (header CTA) / 7×24 (pricing hero)
- One **60px** announcement banner (sub-pill — slightly less round than 9999px to differentiate from CTAs)

Like Cursor and Lovable, Mintlify uses **`lab()` color space as canonical token expression** — Tier 1 evidence of CSS Color Level 4 first-class adoption among modern AI-tooling DS. Padding uses **0.5px increments** (`4.5px`) — sub-pixel precision typical of typography-led DS work.

This corrects the prior footer's claim of "pill nav 9999px / 4×12 / 32px / 15px·500" — accurate but missed:
1. **Mintlify Near-Black** `lab(2.42579 -0.165291 -0.470081)` Primary
2. **Pricing-hero variant** 40px / 7×24 / 16px·500
3. **lab() canonical convention** as DS pattern
4. **60px announcement-banner sub-pill** for "NEW" callouts
