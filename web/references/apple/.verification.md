## Source-domain rule

The Human Interface Guidelines are an official source for platform design guidance. However, the computed CSS of `developer.apple.com/design/human-interface-guidelines/*` describes the documentation website itself. An 18px docs card or 22px docs CTA is not automatically an iOS, macOS, watchOS, tvOS, or visionOS component token.
