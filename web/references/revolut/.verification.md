### Single-system: 9999px Pill + Charcoal/Light-Surface canvas-aware

Revolut uses an **all-pill 42px chrome** with disciplined canvas-awareness:
- **Charcoal `#191c1f` Primary** (consumer light canvas) — warm-cast near-black
- **Light Surface `#f4f4f4` Primary** (Business dark canvas / consumer feature cards) — barely-tinted off-white
- **Translucent `rgba(244,244,244,0.10)`** on-dark hover state
- All radii **9999px** full-pill (no exceptions)
- All heights **42px** (consistent — no compact variant on inspected surfaces; 46px appears on some Open-an-account buttons)
- All weight **500** (medium) — never bold; pill geometry carries weight
- Padding **10×24** standard (10×26 occasional on longer labels)

This is one of the most disciplined single-shape canonical chromes in the corpus — every CTA is 9999px / 42px / 16px·500, only the canvas-pair color flips. Confirms the §11 "fintech super-app, multi-segment, flat" register.

This refines the prior footer's claim of "segment nav 9999px / 8×10 / 38px / 16px·500" — accurate for the segment-nav variant but missed:
1. **Charcoal `#191c1f` Primary** (the consumer-canvas brand color)
2. **Light Surface `#f4f4f4`** Inverse / feature-card CTA
3. **Translucent on-dark** variant
4. Canonical Primary geometry is **42px / 10×24** (slightly larger than segment-nav)
