## Capture fallback added

`web/scripts/capture-reference-evidence.ts` now accepts:

- `--no-interactions` — skip menu/dialog expansion while retaining baseline and pseudo-state capture.
- `--baseline-only` — skip expensive hover/focus enumeration as well, preserving baseline DOM, colors, fonts, typography, spacing, radii, and default component geometry.

This keeps scheduled reverify deterministic on interaction-heavy sites. A baseline-only result never fabricates missing state values.
