### Single-system: 4px Webflow Blue + 8px A11y + 1440px credit pill

Webflow uses **a disciplined 3-radius system**:
- **Primary CTA**: `#146ef5` Webflow Blue **4px** / 51px hero (35px header) / 16-24×8-24 / 16px·**500**
- **A11y skip-nav**: same Blue at **8px** (slightly more rounded for keyboard-focus accessibility)
- **Made-in-Webflow credit pill**: 1440px (essentially full-pill) — only used for the brand-credit indicator
- **Top banner strip**: 0px Webflow Blue (announcement strip — full-width sharp)
- **Webflow Ink `#080808`** for body + nav text — ultra-dark near-black (warmer than `#000`)
- All weight **500** — never bold
- Subpixel padding values (20.775 / 19.177 / 15.9808) suggest **rem-based design tokens** at non-100% root font-size

This refines the prior footer's "nav 3px ghost / 4×10 / 36px / 14px·500; warm dark canvas with agentic-positioning" — accurate for nav variant but missed:
1. **`#146ef5` Webflow Blue Primary** (the brand-defining CTA color)
2. **`#080808` Webflow Ink** (ultra-dark warm-cast near-black for body text)
3. **1440px credit-pill** convention for "Made in Webflow"
4. 4px hero radius (vs 3px nav)
