### Single-system: Light Gray on light + 1000px banner pill + dark-canvas variant

Raycast uses a **subtle three-color Primary**:
- **Primary**: `#e6e6e6` Light Gray + `#2f3031` Charcoal text — 8px / 36px / 14px·500 (the canonical "Download" CTA)
- **Banner pill**: `#130d0e` Deep Black 1000px / 30px (full pill — only for top-promo announcement strip)
- **Dark-canvas Outline**: `rgba(255,255,255,0.05)` 12px (Pro page enterprise CTA)
- **Newsletter inline**: same Light Gray fill but 4px (smaller radius for tight form contexts)
- All weight 500 — never 400, never bold

The **soft pearl-gray Primary** is unusual — most brands use solid black or solid white; Raycast uses a barely-tinted gray (`#e6e6e6`) that fits the macOS prosumer aesthetic without being pure-white. Confirms the §11 "prosumer-aesthetic, keyboard-first" register.

This corrects the prior footer's claim of "nav `#9c9c9d` 6px ghost / 12×8 / 41px / 14px·500" — that's a **darker mid-gray (60% Charcoal)** for nav, but the canonical Primary CTA is `#e6e6e6` 8px (much lighter, brighter). Material correction.
