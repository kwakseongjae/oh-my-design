### Single-system: 9999px Cursor Dark + Cream Pill, 4px content cards

Cursor uses **one consistent button geometry** across home + pricing:
- All Primary/Secondary CTAs `9999px` full-pill (`3.35544e+07px` extreme value confirmed)
- Hero buttons 43px / 16px·400 — Header utility 27px / 14px·400 (same shape, smaller scale)
- Two fills: Cursor Dark `#26251e` (Primary inverse) + Cream `#e6e5e0` (Secondary)
- **Content cards stay 4px** (not pill) — pricing tiers, code editor frames
- Sub-8px padding granularity (`5.6` / `5.88` / `12.48` / `12.8` / `13.4`) confirms the typographic micro-precision claimed in §3

This is one of the tightest single-system documents — the 4-decimal `3.35544e+07px` radius is intentional (theoretically `2^25`, used to guarantee pill rendering at any height).
