### Single-system: 6px Ghost + Voltage Green accent + Sharp tab

VoltAgent uses an **all-ghost hero chrome with sharp-tab navigation**:
- All Primary CTAs: **6px** radius transparent ghost + `#eeeeee` Off-White text — never filled (the brand is OSS-CLI, not enterprise-button)
- **Brand accent**: **`#2fd6a1` Voltage Green** (rgb(47, 214, 161)) — appears on active tabs + search-box text + product accents — sub-`#00d992` lighter shade for search-input variant
- **Tab system**: **0px sharp** with `#101010` Charcoal active background + Voltage Green text (terminal/IDE-aesthetic)
- **Code snippets**: weight 700 monospace + 13px (tight)
- **Search**: 12px on home (more rounded) / 6px on docs (matches CTA radius)
- **Star badge**: 9999px full-pill (the only round chrome — appropriate for a count indicator)
- All weight **700 on Primary** + 500 on nav/tab + 400 on star badge — the **700 weight** is unusual for ghost CTAs (most brands use 500-600); reflects the OSS-engineer "loud-confident" register

This refines the prior footer's "Get Started ghost 6px / 12×16 / 54px / 18px·700; Try VoltOps ghost 6px / 6×12 / 34px / 14px·500" — accurate but missed:
1. **`#2fd6a1` Voltage Green** brand accent (the brand-defining color)
2. **Active tab Charcoal `#101010`** + Voltage Green text pattern
3. **Search input `#00d992`** lighter Voltage Green variant
4. **Star badge 9999px** full-pill GitHub indicator
