### Single-system: Pinkoi Teal search anchor + 100px country pills + 4px content cards

Pinkoi uses an **e-commerce-discovery chrome** with three discrete radii:
- **Search button**: `#10567b` Pinkoi Teal w/ split-radius `0px 8px 8px 0px` (search-box trailing edge — distinctive geometric pattern)
- **Country pills + promo pills**: 100px full-pill (the dominant browse-discovery UI element)
- **Outline secondary**: 4px on white (secondary actions, content cards)
- 3 fills: `#10567b` Teal Primary + `#fff8f7` Coral Tint (active country / promo) + `#eeeeef` Cool Gray (inactive country)
- Charcoal text `#39393e` (warm-cast — slightly purple-leaning vs pure black)
- Weight 500 on chips (slightly weighted up — readability for category browsing)

This is one of the **e-commerce-specific** chrome systems in the corpus — country-pill convention is the signature element (cross-border discovery is Pinkoi's core feature).

This refines the prior footer's claim — `#10567b` 8px is accurate for the search button (with split-radius nuance), but the canonical browse-discovery UI is **100px country pills** with Coral Tint active state — this was undocumented.
