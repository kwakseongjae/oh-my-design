## Font evidence

### Official product-use

MongoDB’s official LeafyGreen typography guidance assigns MongoDB Value Serif to the logo and primarily H1/H2, Euclid Circular A to frequently used product body text, and Source Code Pro to code snippets and other values: https://www.mongodb.design/foundations/typography. This is an official product-design role statement, independent of any particular marketing-page CSS.

### Live computed and FontFaceSet/source corroboration

- **Euclid Circular A** is `loaded`, high confidence, with 287 visible uses across body, buttons, cards, headings, inputs, list items, and text. Computed-family use is corroborated by loaded FontFace data and 19 source URLs, including MongoDB CloudFront WOFF/WOFF2/TTF assets and a MongoDB documentation WOFF file.
- **MongoDB Value Serif** is `loaded`, high confidence, with 5 visible H1/H2 uses. Computed-family use is corroborated by loaded FontFace data and 9 MongoDB CloudFront source URLs for regular, medium, and bold WOFF/WOFF2/TTF assets.
- **Source Code Pro** is `loaded`, high confidence, with 3 visible body/H1 uses. Computed-family use is corroborated by loaded FontFace data and 4 Google-hosted source URLs. Adobe’s upstream repository licenses Source Code Pro under SIL Open Font License 1.1: https://github.com/adobe-fonts/source-code-pro/blob/release/LICENSE.md.
- **Akzidenz-Grotesk Std** is `loaded`, high confidence, with 55 visible body/text uses, but the bundle has no matching source URL and LeafyGreen’s product typography page does not assign it a product role. It remains a documented live observation, not a UI-family token.

### Declared-only, system, and unresolved

ChartsIcons, Font Awesome, FontAwesome4, icomoon, MMSIcons, MMSOrgIcons, and Noto CJK faces are declared with zero visible use. Arial is system use; Times is unresolved due to no matching loaded face. None is promoted or substituted as a MongoDB type token.
