### Single-system: 9999px monochrome white-on-`#000` + dual-context Outline/Inverse split

xAI uses **the most disciplined monochrome system in the corpus** — pure white-on-pure-black with a dual-context Primary split:
- **Home (research-lab register)**: Outline 9999px transparent + `#fff` text
- **/api (developer-product register)**: Inverse 9999px `#fff` bg + `#0a0a0a` text — solid-fill for dense API surface
- **xAI Coal `#0a0a0a`** (slight lift from pure `#000`) for inverse-text — preserves anti-aliasing on pure white
- **Translucent nav `rgba(255,255,255,0.5)`** — half-opacity for inactive hierarchy (vs OpenAI/Anthropic full white)
- All weight **400** uniformly — never bold (research-typography register)
- All radius **9999px** for buttons / 0px for nav — no medium values

### Two-context primary split observation
- This is rare in the corpus — most brands have a single canonical Primary regardless of surface.
- xAI's split signals two product-hierarchy registers: corporate research (outline = restraint) vs developer API (inverse = action).
- Comparable splits: **karrot** (consumer/corporate), **line** (consumer/business), **intercom** (marketing/product) — but xAI's is monochrome rather than dual-color.
