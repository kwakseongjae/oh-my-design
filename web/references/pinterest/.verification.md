### Two-system: Consumer 16px Plum + Business 20-30px Charcoal

Pinterest operates **two distinct chrome systems**:
- **Consumer (kr.pinterest.com)**: Pinterest Red `#e60023` 16px / 48px / 12px·400 + Plum-Black `#211922` ghost nav + Cream `#e5e5e0` Secondary — **the canonical "Pinterest brand chrome"** (warm cream canvas + plum text + red CTA)
- **Business (business.pinterest.com)**: Charcoal `#111111` pill at two radii (20px compact / 30px hero) / 36-60px / 16px·400 — completely different system, more enterprise-tonal

This split is intentional — consumer surface stays in the warm-cream brand register (image-grid first, plum text, red moments-of-action), while Pinterest Business uses a colder enterprise charcoal for B2B advertiser sales.

This corrects the prior footer's "ghost nav 16px / 6×14 / 48px / 12px·400; Plum black `#211922` text confirmed" — accurate but missed:
1. **Pinterest Red `#e60023`** Login Primary
