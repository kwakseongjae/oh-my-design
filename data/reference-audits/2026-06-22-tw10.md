Each brand built by an isolated subagent per `.claude/skills/omd-add-reference/batch-instructions.md`: read SKILL + spec/verification-pipeline + components-schema + regional-sources + stripe/finda exemplars → live-inspect homepage + ≥1 brand-owned surface via Playwright getComputedStyle → Tier-2 probe → write DESIGN.md + .verification.md → self-verify to green. All 10 reached 27/27 on first run.

## Systemic finding

1. **TW brands are invisible to Tier-2 directories** — getdesign.md and styles.refero.design returned **0/10**. Same pattern as KR. All tokens rest on Tier-1 live inspect; the TW regional rule (≥2 brand-owned Tier-1 surfaces) carried every entry.
2. **Google s2 favicon proxy is unreliable for TW domains** — returned 404/generic for ipassmoney, chunghwa. Resolved by pulling the brand's own CDN logo PNG (static01-ipass.cdn.hinet.net, web-eshop.cdn.hinet.net) — a reminder that HiNet-hosted TW brands often need direct-CDN logo sourcing.
3. **Telecom/retail brand color ≠ assumption.** Taiwan Mobile's primary is 活力橘 orange `#ff6700` (not teal); OPENPOINT's digital identity is lavender `#8081ff` (deliberately distinct from 7-ELEVEN's store orange-green). Live inspect corrected two would-be wrong guesses.
4. **Price-red is a market convention.** 591 and iCook-adjacent commerce surfaces use a dedicated red (`#e60012` / `#f04646`) for prices/hot tags — a Taiwan/HK e-commerce idiom worth noting for future TW commerce refs.

## Known shared limitation

Taiwan's most-used finance/payment/loyalty products are **app-first** (iPASS MONEY, EasyWallet, OPENPOINT); their public web is a marketing shell, so in-app token sets are an honest UPDATE target. Telecom self-service portals (Chunghwa) aggressively session-expire deep links, capping second-surface inspection. Process improvement for future TW batches: prefer the brand's **corporate brand-concept / CDN style.css** as the second surface (Taiwan Mobile's `corp.taiwanmobile.com/brand-concept`, OPENPOINT's brand CDN CSS both worked well) over service subpages that session-gate.
